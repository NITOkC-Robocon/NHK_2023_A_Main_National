# NHK_2023_A_Main_National

## 概要
沖縄2023A、獅子舞胴体部のコードです<br>
(※デフォルトブランチ：main)

## 機能
- 肩車機構(前脚の手動昇降)
- ロープ自動送り機能(前脚→中脚→後脚の順に昇降)
  - バックアップ用の手動昇降機能
- お助けアイテム昇降
- お助けアイテム回転

## 環境
|          |     |
|   :-:    | --- |
| プログラム | Arm Keil Studio |
| 使用OS | Mbed OS 5.15.7 |
